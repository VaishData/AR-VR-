

# AR QR Poster Project

An **interactive Augmented Reality poster** built using **AR.js and A-Frame** that displays **3D models when a QR/pattern marker is scanned using a mobile camera**.

This project demonstrates how traditional posters can be transformed into **interactive AR experiences directly in the browser without installing any mobile application**.

# 🚀 Features

* Marker-based **Augmented Reality**
* Interactive **3D models (.glb)**
* Works directly in **mobile browsers**
* No mobile app installation required
* Built using **WebAR technology**
* User interaction using **touch events**
* Redirects users to **Google Forms for interaction**
* Multiple AR experiences using different markers

---

# 🛠 Technologies Used

* **A-Frame** – Web framework for VR/AR experiences
* **AR.js** – Marker-based WebAR library
* **HTML5**
* **JavaScript**
* **GLB 3D Models**
* **Pattern Markers**


# ⚙️ How It Works

1. The user opens the AR webpage on a mobile browser.
2. The browser requests **camera access**.
3. The camera scans the **pattern marker or QR poster**.
4. **AR.js detects the marker**.
5. A **3D model (.glb)** appears on top of the marker in real time.
6. When the user taps the screen, it redirects to a **Google Form or interaction page**.



# 📱 Usage Instructions

1. Host the project on **GitHub Pages or any HTTPS server**.
2. Open the webpage on a **mobile device**.
3. Allow **camera permission**.
4. Point the camera at the **AR marker**.
5. View the **3D augmented object appear on the poster**.
